# 🎟️ Ticket Counting Mechanism

A simple and efficient **Ticket Counting Mechanism** built using **HTML, CSS, and JavaScript**.  
This project helps users count and manage ticket entries dynamically with an interactive user interface.

---

## 🚀 Features

- ✅ Increase ticket count
- ✅ Decrease ticket count
- ✅ Reset counter functionality
- ✅ Real-time count display
- ✅ Simple and responsive UI
- ✅ Beginner-friendly project

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive design
- **JavaScript (ES6)** – Counter functionality and DOM manipulation
